# Quick Start

Get your first "Neural Map" up and running in minutes.

1. **Open DocWeb** in your browser.
2. **Enter a URL** of a documentation site you want to visualize.
3. **Click "Discover"** to start the [Intelligent Discovery](../core-concepts/waterfall-discovery.md) process.
4. **Explore the Graph** using the [Sigma.js Visualization](../features/visualization.md).

Once the graph is populated, you can use the [RAG Chatbot](../features/rag-chatbot.md) to ask questions about the documentation.
