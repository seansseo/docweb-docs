# Installation

To get started with DocWeb, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-repo/docweb.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Set up environment variables**:
   Create a `.env` file with your Firebase and AI configuration.
4. **Run the app**:
   ```bash
   npm run dev
   ```

For more details on the architecture, see [Core Concepts](../core-concepts/index.md).
