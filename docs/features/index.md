# Features

DocWeb offers a suite of tools to visualize and interact with digital content.

## [Sigma.js Visualization](visualization.md)
High-performance, interactive graph rendering using WebGL.

## [RAG Chatbot](rag-chatbot.md)
An AI assistant grounded in your documentation, providing accurate, source-backed answers.

## [Content Extraction](content-extraction.md)
Intelligent scraping and indexing of Markdown data for deep analysis.

Explore how these features are used in our [Use Cases](../use-cases/index.md).
